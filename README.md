# neue fische · Program Portfolio 2026

Internal resources for the Product and Admissions teams. Live on GitHub Pages — open in any browser, no login required.

## [Program Portfolio Deck](https://marielbreuer.github.io/neue-fische/programs_overview-30062026.html)

A slide deck covering all 22 programs across Data & AI, Web & Software Development, Product & Project Management, Cyber & Cloud, UX/UI, and Marketing. Each slide includes target personas, tech stack, job outcomes, and admissions talking points. The final section compares frequently confused program pairs side by side.

Use for: team onboarding, internal alignment meetings, marketing briefings.

---

## [Admissions Assessment Tool](https://marielbreuer.github.io/neue-fische/admissions_assessment_tool.html)

An interactive tool for Admissions Managers to quickly match a prospect to the right program. Select what you know about the learner — background, goal, experience level, time availability, and language — and the tool scores all 22 programs and surfaces the top 3 matches with tailored talking points for the call.

Also includes a persona browser: 30 named archetypes mapped to programs, clickable to pre-fill the assessment form.

Use for: intake calls, lead qualification, team training on program differences.
